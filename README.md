## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic text editor (VS Code, Sublime Text, etc.) for customization

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/larybantu/myteachingportfolio.git
   cd myteachingportfolio
   ```

2. **Open Locally**
   - Simply open `index.html` in your web browser
   - No server required for basic functionality

3. **Deploy to GitHub Pages**
   - Push to your GitHub repository
   - Enable GitHub Pages in repository settings
   - Your portfolio will be live at `https://yourusername.github.io/myteachingportfolio/`

## 🎨 Customization

### Changing Color Scheme

Edit the CSS variables at the top of `css/style.css`:

```css
:root {
    --primary: #1a5276;      /* Main blue color */
    --secondary: #e67e22;    /* Accent orange color */
    --accent: #27ae60;       /* Green accent */
    --text: #333;            /* Main text color */
    --light-bg: #f8f9fa;     /* Light background */
    --border: #ddd;          /* Border color */
}
```

### Adding Your Professional Photo

Replace the emoji placeholder in the About section:

```html
<img src="images/your-photo.jpg" alt="Your Name" class="profile-image">
```

### Updating Content

All sections in `index.html` are clearly marked with HTML comments:

```html
<!-- SECTION_NAME section -->
<!-- Modify content here -->
```

Key sections to customize:
- **Hero Section**: Title, subtitle, and call-to-action
- **About**: Professional bio and teaching philosophy
- **Experience**: Curriculum and teaching highlights
- **Testimonials**: Student feedback and reviews
- **Contact**: Email and social links

### Adding Images

1. Place your images in the `images/` folder
2. Update image paths in `index.html`
3. Optimize images for web performance
